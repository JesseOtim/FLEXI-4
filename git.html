git config --global user.name "Your Name" git config --global user.email "your.email@example.com" Create a GitHub Account: Sign up at github.com. Creating and Cloning Repositories Create a Repository on GitHub: Go to GitHub and click the New button to create a new repository. Fill in the repository name and description, then click Create repository. Clone a Repository: git clone https://github.com/yourusername/your-repository.git Creating Branches Create a New Branch: git branch new-branch Switch to the New Branch: git checkout new-branch Committing and Reverting Commits Make Changes: Edit your files as needed. Stage Changes: git add . Commit Changes: git commit -m "Your commit message" Revert a Commit: git revert Pulling and Pushing Changes Pull Changes from Upstream: git pull origin main Push Changes to Upstream: git push origin new-branch Fetching, Merging, and Renaming Branches Fetch Updates: git fetch origin Merge Branches: git checkout main git merge new-branch Rename a Branch: Working with Pull Requests Create a Pull Request: Go to your repository on GitHub and click New Pull Request. Review a Pull Request: Go to the Pull requests tab. Select a pull request to review. Leave comments and request changes if necessary. Merge a Pull Request: After approval, click Merge pull request. Revert a Pull Request: Click Revert to create a new pull request that undoes the changes. Summary of Common Git Commands # Initialize a new Git repository git init # Clone an existing repository git clone # Create a new branch git branch # Switch to a branch git checkout # Create and switch to a new branch git checkout -b # Stage changes git add git add . # Commit changes git commit -m "Commit message" # Pull changes from upstream git pull origin # Push changes to upstream git push origin # Fetch updates from remote git fetch origin # Merge a branch git merge # Rename a branch git branch -m # Revert a commit git revert
